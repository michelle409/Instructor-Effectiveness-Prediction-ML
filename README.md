# Instructor Effectiveness Prediction using Machine Learning

This project builds a machine learning model to evaluate instructor effectiveness using student engagement and performance metrics.

## Dataset
The dataset contains 2000 records including:

- Completion rate
- Average quiz score
- Score improvement
- Dropout rate
- Watch time
- Assignment submission rate
- Forum activity
- Feedback scores

## Machine Learning Model
Random Forest Classifier was used to classify instructors into:

- Low effectiveness
- Medium effectiveness
- High effectiveness

## Workflow
1. Data exploration
2. Correlation analysis
3. Feature engineering
4. Model training
5. Model evaluation
6. Interpretation of results

## Results
Model accuracy: **~79%**
